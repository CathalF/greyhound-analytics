# Greyhound Racing Value Finder

A web-based application that scrapes greyhound racing data from greyhoundstats.co.uk and live odds from oddschecker, comparing them to identify value betting opportunities.

## Overview

This application combines statistical performance data with current market odds to identify discrepancies where dogs may offer better value than their odds suggest. Users can research races with both datasets side-by-side and receive betting recommendations based on statistical analysis.

## Features

- Daily scraping of greyhound statistics from greyhoundstats.co.uk
- Real-time odds scraping from oddschecker
- Comparison logic to identify value betting opportunities
- Web dashboard for manual research and analysis
- Historical race tracking for pattern identification
