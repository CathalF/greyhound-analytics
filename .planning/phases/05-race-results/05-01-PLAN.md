  <name>Task 1: Add race_results and bet_history tables to schema</name>
  <files>src/storage/schema.sql</files>
  <action>
Add two new tables to schema.sql:

1. race_results table:
   - result_id VARCHAR(100) PRIMARY KEY (format: {race_id}_{position})
   - race_id VARCHAR(100) NOT NULL REFERENCES races(race_id) ON DELETE CASCADE
   - dog_id VARCHAR(100) NOT NULL REFERENCES dogs(dog_id) ON DELETE CASCADE
   - position INTEGER NOT NULL (1-6, 1=winner)
   - finishing_time VARCHAR(20) (e.g., "29.45" seconds)
   - starting_price DECIMAL(10,2) (SP odds at race start)
   - created_at TIMESTAMP DEFAULT NOW()
   - Add index on race_id for efficient lookups

2. bet_history table:
   - bet_id VARCHAR(100) PRIMARY KEY (format: {race_id}_{dog_id}_{timestamp})
   - race_id VARCHAR(100) NOT NULL REFERENCES races(race_id)
   - dog_id VARCHAR(100) NOT NULL REFERENCES dogs(dog_id)
   - suggested_at TIMESTAMP NOT NULL (when value bet was identified)
   - value_score DECIMAL(5,2) NOT NULL (score at time of suggestion)
   - best_odds DECIMAL(10,2) NOT NULL (best odds when suggested)
   - best_bookmaker VARCHAR(50) NOT NULL
   - outcome VARCHAR(20) DEFAULT 'pending' (pending/won/lost)
   - actual_position INTEGER (populated after race)
   - profit_loss DECIMAL(10,2) (calculated: if won = best_odds - 1, if lost = -1)
   - created_at TIMESTAMP DEFAULT NOW()
   - Add index on race_id and outcome

Do NOT modify existing tables. Use CREATE TABLE IF NOT EXISTS for idempotency.
