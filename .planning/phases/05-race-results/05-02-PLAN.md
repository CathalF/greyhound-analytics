<task type="auto">
  <name>Task 1: Create results scraper</name>
  <files>src/scrapers/results_scraper.py</files>
  <action>
Create src/scrapers/results_scraper.py that scrapes race results from oddschecker.

Key implementation:
1. Use base scraper for Playwright/stealth mode (from src/scrapers/base_scraper.py)
2. Navigate to oddschecker greyhound results page: https://www.oddschecker.com/greyhound-racing/results
3. Find completed races by track and time
4. For each completed race:
   - Extract finishing positions (1st through 6th)
   - Extract dog names in finish order
   - Extract SP (starting price) if available
   - Extract finishing time if available
5. Match dog names to existing dog_ids using fuzzy matching (from src/storage/dog_matcher.py)

Functions to implement:
- scrape_results_page() -> List[Dict]: Get all results from today's results page
- scrape_race_result(track: str, race_time: str) -> Optional[Dict]: Get result for specific race
- match_result_to_race(result: Dict, race_id: str) -> List[Dict]: Match scraped dogs to database dogs

Return format for each result:
{
  'track': 'Hove',
  'time': '19:42',
  'positions': [
    {'position': 1, 'dog_name': 'Ballymac Doris', 'sp': 2.5, 'time': '29.45'},
    {'position': 2, 'dog_name': 'Swift Runner', 'sp': 4.0, 'time': '29.67'},
    ...
  ]
}

Handle gracefully: Missing results, different formats, races not found.
  </action>

Create src/results_collector.py - a script to run periodically for collecting results.

Main functions:
1. collect_results():

   - Match to races in database
   - Insert into race_results table
   - Update race status to 'complete'

   - Show betting performance (wins/losses/profit)

CLI interface:
- python src/results_collector.py           # Collect all pending results

- python src/results_collector.py --force   # Re-scrape even if results exist

Intended to run every 30 minutes via cron/scheduler to keep results current.
