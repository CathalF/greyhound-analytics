**Core function: calculate_advanced_value_score()**

Inputs:
- dog_stats: Dict (from dogs table - includes track_stats, distance_stats, recent_form)
- best_decimal_odds: float
- track_name: str (current race track)
- trap_number: int (current trap assignment)
- race_time: datetime (for time-of-day factor)

Output:
- Dict with:
  - total_score: float (weighted composite score)
  - base_score: float (original win_rate / implied_probability)
  - factor_breakdown: Dict of each factor's contribution
  - confidence: str ('high', 'medium', 'low' based on sample sizes)
  - explanation: str (human-readable summary)

**Weighting factors:**

1. **Base Value** (weight: 0.40)
   - Original calculation: win_rate / implied_probability
   - Foundation of the score

2. **Track Factor** (weight: 0.20)
   - From dog_stats['track_stats'], get win_rate for current track
   - Compare to overall win_rate: track_win_rate / overall_win_rate
   - If dog performs 20% better at this track, factor = 1.2
   - Handle missing track data: factor = 1.0 (neutral)

3. **Form Factor** (weight: 0.20)
   - From dog_stats['recent_form'], analyze last 5 races
   - Count positions: wins (1st), places (1st-2nd), in-money (1st-3rd)
   - Strong form (3+ wins in last 5): factor = 1.15
   - Moderate form (1-2 wins): factor = 1.0
   - Weak form (0 wins): factor = 0.85
   - Also detect improving form (recent positions better than earlier)

4. **Trap Bias Factor** (weight: 0.10)
   - Query pattern_analyzer.get_trap_bias() for historical trap win rates
   - Compare this trap's historical win rate to expected (16.67%)
   - If trap 1 has 19% wins: factor = 19 / 16.67 = 1.14
   - Use cached trap bias data (refresh once per session)

5. **Time of Day Factor** (weight: 0.10)
   - From race_time, determine slot (morning/afternoon/evening)
   - Query pattern_analyzer.get_time_of_day_analysis() for ROI by slot
   - If afternoon bets have 15% ROI vs 5% average: factor = 1.10
   - Normalize around 1.0

**Final calculation:**
```python
total_score = (
    base_score * 0.40 +
    base_score * track_factor * 0.20 +
    base_score * form_factor * 0.20 +
    base_score * trap_factor * 0.10 +
    base_score * time_factor * 0.10
)
```

**Confidence level:**
- 'high': All factors have data (track_stats exists for this track, 5+ recent_form entries, trap bias has 50+ samples)
- 'medium': Some factors missing (2-3 factors have data)
- 'low': Mostly missing data (fallback to base score)

**Helper function: get_weighting_factors()**
Returns current factor weights as dict - allows future tuning without code changes.

**Caching:**
- Cache trap_bias and time_analysis for 30 minutes (pattern data doesn't change mid-session)
- Use simple module-level dict with timestamp
