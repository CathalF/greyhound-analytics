Create src/routes/patterns.py with Flask Blueprint for pattern routes.

**Route: GET /patterns**
- Call pattern analyzer functions to get all analysis data
- Pass to template: summary, tracks, value_buckets, trap_bias, time_analysis
- Handle errors gracefully (show message if no data)

**Structure:**
```python
from flask import Blueprint, render_template
from src.services.pattern_analyzer import (
    get_betting_summary,
    get_track_performance,
    get_value_score_performance,
    get_trap_bias,
    get_time_of_day_analysis
)

patterns_bp = Blueprint('patterns', __name__)

@patterns_bp.route('/patterns')
def patterns():
    # Gather all pattern data
    summary = get_betting_summary()
    tracks = get_track_performance()
    value_buckets = get_value_score_performance()
    trap_bias = get_trap_bias()
    time_analysis = get_time_of_day_analysis()

    return render_template('patterns.html',
        summary=summary,
        tracks=tracks,
        value_buckets=value_buckets,
        trap_bias=trap_bias,
        time_analysis=time_analysis
    )
```

**Error handling:**
- If pattern_analyzer functions fail, catch exception and pass empty dict with error message
- Template should handle missing data gracefully
