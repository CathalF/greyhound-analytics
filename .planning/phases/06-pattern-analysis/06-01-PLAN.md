<task type="auto">
  <name>Task 1: Create pattern analyzer service</name>
  <files>src/services/pattern_analyzer.py</files>
  <action>
Create src/services/pattern_analyzer.py with comprehensive pattern analysis functions.

**Core analysis functions:**

1. get_track_performance() -> Dict[str, Dict]:
   - Query bet_history joined with race_results grouped by track
   - For each track: win_count, loss_count, win_rate, total_profit, roi
   - Return dict keyed by track name

2. get_value_score_performance(buckets: List[Tuple[float, float]]) -> Dict[str, Dict]:
   - Analyze bet outcomes by value score ranges
   - Default buckets: [(1.2, 1.3), (1.3, 1.4), (1.4, 1.5), (1.5, 2.0), (2.0, float('inf'))]
   - For each bucket: win_count, loss_count, win_rate, avg_odds, total_profit, roi
   - Helps identify optimal value threshold

3. get_form_correlation() -> Dict:
   - Analyze correlation between recent form and bet outcomes
   - Group by recent_form patterns (e.g., dogs with 3+ wins in last 5 races)
   - Return: form_category -> {win_rate, sample_size, avg_value_score}

4. get_trap_bias() -> Dict[int, Dict]:
   - Analyze win rates by trap number (1-6)
   - From race_results: count wins by trap position across all races
   - Return: trap_number -> {wins, total_races, win_rate}

5. get_time_of_day_analysis() -> Dict[str, Dict]:
   - Group races by time slot (morning, afternoon, evening)
   - morning: before 14:00, afternoon: 14:00-18:00, evening: after 18:00
   - Analyze bet performance in each slot

6. get_betting_summary() -> Dict:
   - Overall stats: total_bets, wins, losses, win_rate, total_profit, roi
   - Best performing track, worst performing track
   - Best value score bucket
   - Streak info: current_streak (win/loss count)

**Implementation notes:**
- Use parameterized SQL queries via get_db()
- Handle empty datasets gracefully (return sensible defaults)
- Round percentages to 2 decimal places
- Include sample_size in all aggregations for statistical context
- Follow patterns from src/storage/race_results.py for database access
